# onboarding-assignment

https://onboarding-assignment-orpin.vercel.app/

회원가입 로그인 기능, TODO LIST 조회, 정보 수정이 가능한 웹 페이지입니다.

## 기술 스택

- React
- TypeScript
- Tanstack Query
- zustand
- Tailwind CSS
- Axios
- React Router DOM
